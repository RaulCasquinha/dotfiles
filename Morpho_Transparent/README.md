# This is a theme that changes with wallpaper

## Wallpaper switching

This is done with swww, just call 'matugen image path/to/image',
alternatively you can use [Waypaper](#waypaper) for this

## Matugen

Matugen is a program that extracts colors from an images and generates a color palette

## Waypaper

Waypaper is a program to switch wallpapers,
you can add a post script that calls Matugen automatically
